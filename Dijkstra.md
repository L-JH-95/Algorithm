## Dijkstra vs BFS

##### 기본적으로는 같은 bfs 방법 사용 (queue)

|                 | Dijkstra          | Bfs     |
| --------------- | ----------------- | ------- |
| 간선 길이       | 클 때             | 작을 때 |
| 가중치          | 다를 때           | 같을 때 |
| 중복            | 가능  (조건 존재) | 불가능  |
| visit 초기화 값 | INF (2147000000)  | 0/-1    |
